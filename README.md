## ⚙️ How to Run Projects

1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repo-name.git
```
2. Navigate to a project folder:
```bash
cd project-folder-name
```

3. Install dependencies:
```bash
npm install
```

5. Run the project:
```bash
npm start
```
